Tinychat plugin for Wordpress. This plugin adds webcam chat support to your wordpress blog / site.. easy click install.
All you need to do is download, active and configure!

== Description ==

Tinychat offers and webchat client supporting several hundre of users. Also support multiple users broadcasting
webcam and private chat. This plugin incorporates this in to your wordpress blogg / site and gives your visitors
the ability chat, and share there experiance of your site!

* FREE!
* Webcam
* Private Chat
* Text chat
* 100's of users
* FREE!
* Can limit to registerd users
* Select Moderator roles
* Easy to use!

Feel free to report any and all bugs you may find!

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

First:
1. go to www.tinychat.com
2. Footer Menu -> API
3. Register
4. Click 'Develop apps with tinychat'
5. Select and nice App name! Maybe your site name ?
6. Describe your site, and enter the url to your site!
7. Get your api key and secret!

1. Upload `tinychat` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the TInychat Admin menu on the bottom of your admin menu
4. Fill in all information needed
5. Dont forget to select enabled!
6. Go to the widget menu, and add the tinychat widget to your sidebar!
